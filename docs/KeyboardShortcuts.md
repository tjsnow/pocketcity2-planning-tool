# Keyboard Shortcuts

| Key | Action |
| --- | --- |
| V | Select tool |
| B | Bulldozer |
| G | Toggle grid layer |
| W | Toggle warning layer |
| R | Rotate selected building |
| Arrow keys | Move selected building |
| Escape | Return to Select |
| Ctrl/Cmd+Z | Undo |
| Ctrl/Cmd+Shift+Z | Redo |

Shortcuts are ignored while typing in text, search, and select controls.
